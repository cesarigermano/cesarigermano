# Germano Cesari

Senior C++ engineer based near Frankfurt, Germany, with long experience in performance-critical native software, graphics/GPU systems, 3D/DCC tools, geometry processing, debugging, profiling, and large production codebases.

Currently Senior ZBrush Developer at Maxon, after initially joining Maxon as part of the Redshift renderer team.

Most of my professional work is proprietary. This profile highlights selected independent C++/3D tooling projects through public descriptions and demo material.
